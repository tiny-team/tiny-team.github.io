First post

2020-12-08

Hello! This is Animesh. I have started this org tiny-team with an ex-colleague and friend Sai Raghava. Both of us are curious and passionate developers. Over the years, we have made a few attempts to collaborate on software projects. This is one of those attempts again and the most promising one so far.

As a first project, we are working on a lightweight and self-hosted kanban application inspired by wekan, kanboard and such projects. We intend to dogfood this project fully and develop into a decent self-hosted alternative for developers.

I am a full stack web developer with focus on ASP.NET Core, Angular and SQL Server, while Sai is a Backend developer with focus on Java and Microservices.
